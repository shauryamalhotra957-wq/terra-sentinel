# Contributing

Thanks for improving Terra Sentinel.

## Development setup

1. Fork or clone the repository.
2. Create a focused branch from `main`.
3. Install the locked dependencies with `npm ci`.
4. Start the app with `npm run dev`.

## Quality checks

Run the project verification suite:

```bash
npm run check
```

Update tests and documentation with behavior changes. Include visual evidence when a map, chart, or responsive layout changes.

## Pull requests

Explain the problem, implementation, and validation. Keep changes focused and do not commit secrets or generated output.
