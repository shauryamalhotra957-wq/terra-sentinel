# Research Sources

The project concept was shaped by these official/public references:

- UNDRR/WMO, `Global Status of Multi-Hazard Early Warning Systems 2025`: https://www.undrr.org/reports/global-status-mhews-2025
- United Nations, `Early Warnings for All`: https://www.un.org/en/climatechange/early-warnings-for-all
- FEMA, `Community Lifelines`: https://www.fema.gov/emergency-managers/practitioners/lifelines
- FEMA, `Community Lifelines Implementation Toolkit`: https://www.fema.gov/emergency-managers/practitioners/lifelines-toolkit
- WHO, `Health Emergency and Disaster Risk Management Framework`: https://www.who.int/publications-detail-redirect/health-emergency-and-disaster-risk-management-framework
- NOAA NCEI, `U.S. Billion-Dollar Weather and Climate Disasters`: https://www.ncei.noaa.gov/access/billions/

These sources informed the framing, not the synthetic dataset. No source data was copied into the application.
